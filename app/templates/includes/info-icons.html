How to use it

Learn to build this amazing app in a few simple steps.

Get the code To build this app, you must be familiar working with the console and Python, Flask must be installed on your system. $ git clone https://github.com/app-generator/flask-bulma-css.git

Add your magic This app has a small footprint makes the development a breeze. The whole project is well documented, to help any developer to start fast. Access here the README file, shipped with the sources.

Deploy The project can be exported as a static app. Just type $ python ./static.py and the whole praject become a blazing fast app. Read more about this app, on a dedicated blog article:
Adding Bulma CSS to Flask
